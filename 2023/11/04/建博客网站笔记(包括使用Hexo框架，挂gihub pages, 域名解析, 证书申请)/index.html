建博客网站笔记(包括使用Hexo框架，挂gihub pages, 域名解析, 证书申请)

Created2023-11-04|Updated2023-11-04
|Post Views:

建站第一篇，记录下建站流程。基本上按照别人写的教程来的，我就写个大概，忘掉的时候用来回忆。

本地准备

下载nodejs, git，Hexo

npm install -g hexo-cli

在选定的文件夹里配置hexo

cd xxx
hexo init
hexo s

打开链接看效果
然后安装一个好看的主题和插件
git clone -b master https://gitee.com/immyw/hexo-theme-butterfly.git themes/butterfly
npm install hexo-renderer-pug hexo-renderer-stylus –save
hexo clean //执行此命令后继续下一条
hexo g //生成博客目录
hexo s //本地预览
打开链接看效果

管理本地/远程代码

直接用github建一个新仓库push

图床

Picgo配合github仓库，网上有很多资料

GitHub Pages

准备将网站挂在在github pages上
page的建立这个网上也有很多教程，保证仓库是public的

部署网站到pages

修改.config.yml里depoy选项
hexo clean //执行此命令后继续下一条
hexo g //生成博客目录
hexo s //本地预览
hexo d //部署项目

打开链接看效果(可能要等GitHub反应一会)

搞域名

域名买国内的需要备案，海外的不备案
我买的是namesilo服务商的
买好后去管理界面-》DNS管理-》用github模板就
然后在本地的source里新建一个CNMAE文件，内容填域名，再次部署 （只有source里的文件会被原封不动的上传）
完成！
打开链接看效果(可能要等域名反应一会)

加证书

github pages应该是不用加的，但如何使用其他服务器建站可能就需要，这里记录一下
现在载入网站会提示证书不正确一类的，因为就没添加证书
网上有免费证书，还有一键式添加/更新服务的开源项目，这里直接使用
https://github.com/acmesh-official/acme.sh

git clone https://github.com/acmesh-official/acme.sh.git
cd ./acme.sh
./acme.sh --install -m my@example.com #这里修改成你自己邮箱

在域名服务商那儿取得密钥，自动API颁发证书，namesilo是在API manager里申请即可

export Namesilo_Key="<key>"
./acme.sh --issue --dns dns_namesilo -d example.com -d *.example.com --dnssleep 900

漫长的等待，去干点别的事吧
检查证书问题是否解决
Done
